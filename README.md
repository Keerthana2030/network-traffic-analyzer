# 🚀 Network Traffic Analyzer

## 📌 Overview

This project is a Python-based Network Traffic Analyzer built using Scapy. It captures and analyzes real-time network packets to understand traffic patterns and gain insights into network behavior.

> 💡 This project can be extended into an AI-powered Intrusion Detection System (IDS).

---

## 🚀 Features

* 📡 Real-time packet sniffing
* 🌐 Protocol detection (TCP, UDP, ICMP)
* 🔍 Source & destination IP tracking
* 📊 Traffic inspection and logging
* ⚡ Lightweight and fast analysis

---

## 🛠️ Tech Stack

* Python
* Scapy
* Pandas
* Matplotlib

---

## ⚙️ How It Works

1. Captures live packets using Scapy
2. Extracts important information:

   * Source IP
   * Destination IP
   * Protocol type
3. Processes and displays structured output
4. Helps analyze network communication patterns

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
sudo python3 network_traffic_analyzer.py
```

> ⚠️ Note: Requires administrator/root privileges for packet capture.
